# Mucyan 沐沧科技官网

沐沧科技（淄博）有限公司品牌官网，使用 Vue 3、TypeScript、Vite 与 pnpm 构建。

## 本地开发

```bash
pnpm install
pnpm dev
```

## 构建

```bash
pnpm build
```

构建产物位于 `dist/`。`dist/client` 为静态资源，`dist/server/index.js` 为部署入口。

## 发布前需要确认
